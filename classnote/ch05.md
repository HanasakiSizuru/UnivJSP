# 05 JSP 기본 문법
>## 01. 주석
---
```jsp
<!--(주석)--> : HTML주석
<%--(주석)==> : JSP주석 -> .jsp에서만 동작
```
>## 02. 지시어
---
### <%@  &nbsp;&nbsp;&nbsp; %>
### **1. Page**(중간고사)
```jsp
⭐⭐<%@ page contentType="text/html; Charset=UTF-8" import%> - (항상 사용)⭐⭐
<%@ page import="java.util.*" %> - (Java import와 같음)
```
>## 03. 액션
---
```jsp
<jsp: (actionname) (attribute)="(value)">
```
